# 🍟 PHẦN MỀM QUẢN LÝ CỬA HÀNG ĐỒ ĂN VẶT
## 📖 Giới thiệu

Dự án **Phần mềm quản lý cửa hàng bán đồ ăn vặt** được xây dựng nhằm hỗ trợ việc quản lý bán hàng, sản phẩm và doanh thu một cách hiệu quả, nhanh chóng và chính xác.

Phần mềm giúp cửa hàng:
* Quản lý danh sách sản phẩm
* Theo dõi đơn hàng
* Kiểm soát doanh thu
* Tiết kiệm thời gian và công sức

## 🛠️ Công nghệ sử dụng

## 📂 Cấu trúc thư mục

```

* Quản lý sản phẩm (thêm, sửa, xóa)
* Quản lý đơn hàng
* Tính tiền và in hóa đơn
* Thống kê doanh thu
* Tìm kiếm sản phẩm
